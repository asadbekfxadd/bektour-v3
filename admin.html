Управление фото и видео сайта
Токен сохраняется только в этом браузере (localStorage) и никуда не отправляется, кроме api.github.com.
Как создать: github.com → Settings → Developer settings → Fine-grained tokens → Generate new token → Repository access: Only select repositories → bektour-v3 → Permissions → Contents: Read and write → Generate.

Номер присвоится автоматически. Фото сжимаются до 1920px.
Порядок на сайте = порядок номеров. При удалении файла из середины остальные автоматически перенумеруются, чтобы не было пропусков.
